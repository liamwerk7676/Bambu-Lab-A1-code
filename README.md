# Bambu-Lab-A1-code

A repository containing custom modified G-code for the Bambu Lab A1. This includes scripts and optimizations developed by me, alongside community-sourced logic.

## Usage
These scripts are optimized for **OrcaSlicer**. 

1. Open OrcaSlicer.
2. Navigate to **Printer Settings** -> **Machine G-code**.
3. Replace the existing start, end, or filament G-code with the versions found in this repo.
4. Always monitor the first layer of your print when testing new G-code.

## Features
* Modified start sequences for faster bed leveling.
* Custom end G-code for better nozzle wiping and part cooling.
* Specific tweaks for the A1's sensor calibration.
